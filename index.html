AI / ML Course — Overview

This documentation site contains unit-wise notes, practical assignments, datasets, suggested readings and cohort information for the AI / ML Course.

Course Abstract

This course introduces machine learning fundamentals, supervised and unsupervised techniques, model evaluation, and real-world projects. The objective is to provide both theoretical foundations and hands-on skills with Python, scikit-learn, and basic deep learning concepts.

Quick Start

  1. Clone this repository (or download the zip) to view source notes.
  2. Each Module page contains slides, code snippets, and recommended datasets.
  3. Assignments page lists deliverables and evaluation rubrics.

Learning Outcomes

  • Understand supervised & unsupervised algorithms
  • Evaluate models and avoid overfitting
  • Implement basic neural networks for classification
